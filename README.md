# 🛒 E-commerce React App

A modern and responsive **E-commerce Web Application** built using React and Tailwind CSS.  
It provides a smooth shopping experience with product browsing, category filtering, and cart management.

## 🚀 Features

### 🛍️ Product Browsing
- View products with images, names, and prices  
- Clean and user-friendly layout  

### 🔎 Category Filtering
- Browse products by:
  - Men  
  - Women  
  - Kids  

### 🛒 Cart Management
- Add items to cart  
- Remove items from cart  
- Increase / decrease quantity  
- Real-time updates  

### 💾 Persistent Cart
- Cart data stored using **localStorage**  
- Items remain after page refresh  

### 🔔 Toast Notifications
- Instant feedback using **react-hot-toast**  
- Add/remove item alerts  

### 📱 Responsive Design
- Fully responsive across:
  - Mobile  
  - Tablet  
  - Desktop
